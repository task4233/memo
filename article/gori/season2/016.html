# B - 回転

# 概要

N×NN \times NN×Nの文字列を90度回転させて出力せよ.

# 解法

  • ↓→\color{red}{\downarrow}\color{blue}{\rightarrow}↓→の入力を↓←\color{blue}{\downarrow}\color{red}{\leftarrow}↓←に変換してやれば良い.
  • 出力部分は以下の通り.
for(int wi=0;wi<n;++wi){
  for(int hi=n-1;hi>=0;++hi){
    cout << s[hi][wi];
  } 
  cout << endl;
}

# D - 塗り絵

# 概要

  • NNN個の島がN−1N-1N−1本の橋で繋がっている.
  • この時, 各島を白と黒を使って色塗りする時の塗り方は何通りか?
  • ただし, 橋の両端の島が黒になってはならない.

# 解法

Editorialに沿って解く.

NNN頂点, N−1N-1N−1辺なので, このグラフは木であることがわかる.
その上で, 木の任意の頂点の通り数を, その頂点の子の通り数を用いて表したい(∵\because∵ DPの漸化式を作りたい)

f(x),g(x)f(x), g(x)f(x),g(x)を以下のように定義する.

f(x):=f(x) :=f(x):= 頂点xxxを根とする部分木を白と黒で塗る時の通り数.
(ここで, 頂点xxxは白でも黒でも良い)

g(x):=g(x) :=g(x):= 頂点xxxを根とする部分木を白と黒で塗る時の通り数. (ただし, 頂点xxxは黒でなければならない)

# 頂点xxxが白の時

頂点xxxが白だった時を考える.
頂点xxxが白の時, その子は白でも黒でも良い.

したがって, 頂点xxxの子をy1,...,yky_1, ..., y_ky1​,...,yk​とすると,
この時の頂点xxxの通り数は, ∏i=1kf(yi)\displaystyle{\prod_{i=1}^{k} f(y_i)}i=1∏k​f(yi​)となる.
この式の意味は, 具体的に書き出してみれば分かるので, そこは紙にでも書いてみると良いと思う.

# 頂点xxxが黒の時

頂点xxxが黒だった時を考える.
頂点xxxが黒の時, その子は白でなければならない(∵\because∵ 題意).

したがって, 頂点xxxの子をy1,...,yky_1, ..., y_ky1​,...,yk​とすると,
この時の頂点xxxの通り数は, ∏i=1kg(yi)\displaystyle{\prod_{i=1}^{k} g(y_i)}i=1∏k​g(yi​)となる.

# 一般化

この上で, 頂点xxxの色の指定がないf(x)f(x)f(x)は以下のように定義できる.
f(x)=∏i=1kf(yi)+∏1kg(yi)f(x) = \displaystyle{ \prod_{i=1}^{k} f(y_i) + \prod_{1}^{k} g(y_i)}f(x)=i=1∏k​f(yi​)+1∏k​g(yi​)

また, 頂点xxxが黒と指定されているg(x)g(x)g(x)は, 子が全て白と確定しているため, 以下のように定義できる.
g(x)=∏i=1kf(xi)g(x) = \displaystyle{ \prod_{i=1}^{k} f(x_i) }g(x)=i=1∏k​f(xi​)

g(x)g(x)g(x)を用いると, $f(x)はさらに短くでき, 一般化した式は以下のようになる.

  • f(x)=g(x)+∏i=1kg(yi)f(x) = \displaystyle{ g(x) + \prod_{i=1}^{k} g(y_i)}f(x)=g(x)+i=1∏k​g(yi​)
  • g(x)=∏i=1kf(xi)g(x) = \displaystyle{ \prod_{i=1}^{k} f(x_i) }g(x)=i=1∏k​f(xi​)

これを実装すればよい.
